Enterprise Windows Server 2025 AD & SIEM Lab
VirtualBox · Windows Server 2025 · Windows 11 · Splunk Enterprise
Completed

Built a fully functional enterprise-grade Active Directory environment from scratch using VirtualBox. Deployed Windows Server 2025 as a Domain Controller (DC01), a Windows 11 domain client (WIN11-01), and a dedicated Splunk SIEM server (SPLUNK01). Configured AD DS, DNS, DHCP, Group Policy, NTFS permissions, Sysmon log forwarding, and Splunk dashboards — replicating real corporate IT infrastructure and SOC monitoring workflows end-to-end.

Objectives
  • Deploy and administer a Windows Server 2025 Active Directory domain from scratch
  • Configure DNS, DHCP, Group Policy, and NTFS permissions mirroring a real corporate environment
  • Provision users and security groups at scale using PowerShell automation
  • Set up a dedicated Splunk SIEM to ingest and monitor Windows event logs and Sysmon telemetry
  • Simulate real security events and verify detection in Splunk dashboards
  • Build a portfolio-ready lab with documentation, screenshots, and a structured GitHub repo
What Was Built
  • Deployed DC01 — Windows Server 2025 Domain Controller with AD DS, DNS, and DHCP fully configured
  • Created domain blaborte.local with 8 OUs: IT, HR, Finance, Management, Servers, Workstations, Service Accounts, Security Groups
  • Created 4 Global Security Groups (IT-Users, HR-Users, Finance-Users, Management-Users)
  • Provisioned 48 domain users via PowerShell bulk script — auto-assigned to OUs and Security Groups with forced password change at next logon
  • Configured DHCP scope 192.168.1.100–200 with DNS and domain options pushed to clients
  • Deployed WIN11-01 — Windows 11 client joined to the domain, tested login as BLABORTE\IT.user1
  • Created 12 Group Policy Objects across security, user environment, and device restriction categories
  • Built D:\Departments shared folder structure with NTFS permissions scoped per Security Group (RBAC)
  • Deployed SPLUNK01 on Ubuntu Server 24.04 with Splunk Enterprise accessible at port 8000
  • Installed Sysmon on DC01 and WIN11-01 with SwiftOnSecurity community config
  • Installed Splunk Universal Forwarder on both Windows VMs forwarding Security, System, and Sysmon logs
  • Built Splunk dashboards for Authentication Monitoring, PowerShell Activity, and Endpoint Monitoring
  • Simulated brute-force logins, account lockouts, and PowerShell execution — verified all detected in Splunk
Key Learnings
  • Deep hands-on experience with Windows Server 2025 administration — AD DS, DNS, DHCP, and file server roles in a live domain
  • Understood how Group Policy is structured: domain-level security vs. OU-level user and device policies, and how inheritance works
  • Practiced PowerShell automation for bulk user provisioning, group management, and NTFS permission scripting
  • Learned how Sysmon enriches Windows event logging — the difference between raw Security logs and enriched process/network telemetry
  • Built real Splunk SPL query skills: filtering by EventCode, stats, timechart, and saving panels to dashboards
  • Experienced the defender's perspective — simulating attacks and watching them surface in a SIEM in real time
  • Understood NTFS and RBAC permissions in depth by testing access as different department users across shared folders
